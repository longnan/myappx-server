# MyAppx Server All-in-One

The all-in-one solution to build and run MyAppx Server in Docker.

## Installation & Management

### Docker Desktop for Windows

| Action | Command |
|--------|---------|
| **Install** | `setup.bat` |
| **Uninstall** | `cleanup.bat` |
| **Update** | `update.bat` |

## Services & Links

### MyAppx Services

- **MyAppx Home**
  - URL: [https://localhost](https://localhost)
  - Credentials: `superuser@local.corp` / `System`

- **Server Monitor**
  - URL: [https://localhost/idempiereMonitor](https://localhost/idempiereMonitor)

- **Interface Service**
  - URL: [https://localhost/ADInterface/services](https://localhost/ADInterface/services)

- **Felix Web Console**
  - URL: [https://localhost/osgi/system/console/bundles](https://localhost/osgi/system/console/bundles)

### Management Tools

- **Portainer**
  - URL: [https://localhost/portainer/](https://localhost/portainer/)
  - Credentials: `admin` / `portainer`

- **PGAdmin**
  - URL: [https://localhost/pgadmin4/](https://localhost/pgadmin4/)
  - Credentials: `pgadmin4@local.host` / `pgadmin4`
